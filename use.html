install animateMePlz

Get animateMePlz up and running in under a minute. Hardly any javascript, jQuery or CSS knowledge necessary.

Download

1. Insert jQuery & animateMePlz.js

Once you have downloaded the necessary files should place the animateMePlz.js file in the head of your HTML file. It should go below jQuery. 


<head>
...
<!-- insert jQuery (you can also download the file and insert it) -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
<!-- insert animateMePlz.js into your html -->
<script src="path/to/animateMePlz.js" type="text/javascript"></script>
</head>

2. Initiate animateMePlz

Add a script tag below the first two that initiates your function on all the elements with the class "animated". You'll see why we use the class animated in step 4. 


<head>
...
<!-- insert jQuery (you can also download the file and insert it) -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
<!-- insert animateMePlz.js into your html -->
<script src="path/to/animateMePlz.js" type="text/javascript"></script>
<!-- initiate function on scroll -->
<script>
    $(document).ready(function(){
        $(window).scroll(function() {
            $('.animated').animateMePlz();
        });
    }) 
</script>
</head>

Note: If you only want to animate elements with a certain class in (addition to animated), such as only elements that will fade in, you can replace $('.animated') with $('.animated.fadeIn'). 


<head>
...
<!-- initiate function on scroll -->
<script>
    $(document).ready(function(){
        $(window).scroll(function() {
            $('.animated.fadeIn').animateMePlz();
        });
    }) 
</script>
</head>

Say that instead of animating every element with the class '.animated', you only want to animate elements with the class '.thisDiv'. In that case, you would substitute $('.animated') with $('.animated.thisDiv').

3. Insert animate.css

You can insert animate.css via CDN by copying what I entered below. Make sure this is in the <head> of your document. You an also download the file from Dan Eder's website. 


<head>
...
<!-- insert jQuery (you can also download the file and insert it) -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
<!-- insert animateMePlz.js into your html -->
<script src="path/to/animateMePlz.js" type="text/javascript"></script>
<!-- initiate function on scroll -->
<script>
    $(document).ready(function(){
        $(window).scroll(function() {
            $('.animated').animateMePlz();
        });
    }) 
</script>
<!-- animate.css -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/3.5.2/animate.min.css">
</head>

4. Apply to elements

Add the class 'animated' to any element you want to animate. Separately, add a class for the animation style you want to use ('fadeIn', for example.) Go to Dan Eder's website to see all available animations. 


<body>
...
<!-- insert jQuery (you can also download the file and insert it) -->
<div class="animated fadeIn">
    Fade this div in!
</div>
</body>
